## 1. 简介

| 功能 | 说明 |
|------|------|
| ⏱️ 专注计时器 | 番茄钟计时，提高专注力 |

| 📅 任务规划 | 主线/支线任务管理 |

## 2. 功能使用指南

### 2.1 专注计时器（首页）

**功能入口**：首页默认显示

**使用步骤**：

1. **选择时长**：点击顶部时长按钮（15/25/45/60分钟）
2. **选择心情**：开始前选择当前心情状态
3. **开始计时**：点击「开始」按钮
4. **暂停/继续**：点击「暂停」按钮可暂停计时
5. **完成记录**：计时结束后可对本次专注进行评分

**界面说明**：
- 圆形进度环：显示剩余时间比例
- 数字显示：当前剩余时间（MM:SS）
- 底部统计：今日专注时长和完成番茄数

---

### 2.2 AI 树洞

**功能入口**：底部导航栏「树洞」图标

**使用步骤**：

1. **进入聊天**：点击底部导航「树洞」
2. **输入消息**：在输入框中输入想说的话
3. **选择情绪标签**：点击快捷情绪标签快速发送
4. **发送消息**：点击发送按钮或按回车键
5. **查看回复**：AI 会根据情绪给出共情回复

**情绪标签**：
- 😢 难过
- 😕 焦虑
- 😐 平静
- 🙂 开心
- 😄 很开心

---

### 2.3 任务规划

**功能入口**：底部导航栏「任务」图标

**使用步骤**：

1. **创建主线任务**：
   - 点击「添加主线」按钮
   - 输入任务名称和截止日期
   - 选择颜色标签

2. **创建支线任务**：
   - 在主线任务下点击「添加支线」
   - 输入任务名称和截止日期
   - 可设置提醒时间

3. **完成任务**：
   - 点击任务前的复选框标记完成
   - 完成后会自动记录心情

**任务类型**：
- 🔴 主线任务：长期目标，显示进度条
- 🔵 支线任务：单日目标，可勾选完成

---

### 2.4 情绪日历

**功能入口**：底部导航栏「日历」图标

**使用步骤**：

1. **查看月历**：默认显示当月日历
2. **查看详情**：点击任意日期查看当天记录
3. **查看趋势**：底部显示最近30天情绪趋势图

**颜色说明**：
- 🔴 红色背景（1-3分）：心情较差
- 🟡 黄色背景（4-6分）：心情一般
- 🟢 绿色背景（7-10分）：心情较好

---

### 2.5 点滴日志

**功能入口**：底部导航栏「日志」图标

**使用步骤**：

1. **记录日志**：点击「+ 记录」按钮
2. **选择心情**：选择当前心情表情
3. **输入内容**：记录发生的事情（可选）
4. **添加图片**：可上传图片记录（可选）
5. **保存日志**：点击保存按钮

**时间轴**：日志按时间倒序排列，最新的显示在最上方

### 2.6 治愈电台

**功能入口**：底部固定播放器

**使用步骤**：

1. **选择音效**：点击播放器图标展开选择
2. **选择类型**：雨声/森林/咖啡馆/海浪/篝火/鸟鸣
3. **调节音量**：拖动音量滑块
4. **播放/暂停**：点击播放按钮

**音效类型**：
- 🌧️ 雨声：适合专注学习
- 🌲 森林：放松心情
- ☕ 咖啡馆：增加氛围感
- 🌊 海浪：助眠放松
- 🔥 篝火：温暖治愈
- 🐦 鸟鸣：清新自然

## 4. 常见问题

### Q1：数据会保存在哪里？

**A**：所有数据都保存在您的浏览器本地（localStorage），不会上传到服务器，保护您的隐私。

### Q2：换设备后数据会丢失吗？

**A**：由于数据存储在本地，换设备后数据不会自动同步。建议定期导出数据备份。

### Q3：如何删除数据？

**A**：清除浏览器缓存或使用个人中心的「清除数据」功能（如已添加）。

### Q4：为什么手机访问不了？

**A**：如果部署在国外平台（如 Vercel），可能需要切换网络（如使用流量）或等待国内 CDN 生效。

### Q5：番茄钟响铃不生效？

**A**：请确保浏览器允许通知权限，并开启声音。
